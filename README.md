## Zeros NFT ID

Mint an NFT with an NFT ID starting with >= 2 leading zeros in the ID.
Hex encoded NFT ID must start with `0x0000`, example NFT ID `0x000000e30408fd57bf0b8f556dac652df068469d34fcaed1b1a1db24741285e8`.
